# Todo Application

A simple and elegant todo application built with HTML, Tailwind CSS, and JavaScript.

## Features

- ✅ Add new tasks with title and description
- ✅ Mark tasks as completed/incomplete
- ✅ View task details
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Persistent storage using localStorage
- ✅ Responsive design
- ✅ Clean and modern UI

## Local Development

1. Place this folder in your XAMPP htdocs directory
2. Start Apache from XAMPP Control Panel
3. Open `http://localhost/to_do` in your browser

## Deployment

This app can be deployed to Vercel, Netlify, or any static hosting service.
